# Regra agendamento

O usuário precisa selecionar uma estabelecimento (barbearia, salao...)
O usuário precisa selecionar um servico
O usuário precisa verificar o servico desejado
O usuário precisa escolher um horário
Se não existir um horario disponivel o usuario deve selecionar um horario disponivel
Se o existir um horário disponivel
O usuário seleciona o horario para agendamento
